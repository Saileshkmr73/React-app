Steps to install React in Windows:

1.Download Node.js from the link : https://nodejs.org/en/download/
2. Install Node.js with all recommended steps
3.Open Command Prompt and give the  command: npm install create-react-app -g (note: dont forget to add -g as it install the react globally)
4. Once Installed create the react build folder on your desired path by giving below command: create-react-app [repository name] (Ex: c:\Sailesh>create-react-app react-complete-guide)
5. Once install run the development server by giving command: npm start
6. A development server will be opened in the browser. Dont close the browser and start working with js files and check the browser with updated changes\
